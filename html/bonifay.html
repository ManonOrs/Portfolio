API Bonifay

Création d'une API pour l'entreprise Bonifay materiaux de construction

L'ors d'un stage de sept semaines dans l'entreprise Bonifay, j'ai eu pour mission de crée une API pour l'entreprise qui elle meme communiquais avec les API de Wayzz. Cette API gerrais la géolacalisations des camions, le dépassements de vitesses d'un camion ou la vitesse d'un camions dans une zone donnée

Technologies utilisées

  • Php
  • Html/Css/Javascript
  • Framework Symfony
  • mailjet
  • Cron
  • Git
